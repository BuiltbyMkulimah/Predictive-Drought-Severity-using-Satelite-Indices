This project is a sophisticated geospatial pipeline designed to monitor drought conditions in East Africa by integrating satellite-derived data, performing automated spatial alignment, and training machine learning models to classify drought severity.🌍 East Africa Drought Monitoring Pipeline🚀 OverviewThis project automates the transformation of raw satellite imagery (Precipitation and Vegetation Health data) into actionable intelligence. By standardizing geospatial grids and applying machine learning, the system identifies drought patterns across diverse regions, providing a scalable solution for environmental analysis.🛠 Key FeaturesAutomated Data Extraction: Seamlessly handles multiple country datasets.Geospatial Harmonization: Uses rasterio and gdal to reproject and resample heterogeneous raster grids into a unified spatial resolution.Data Cleaning & Masking: Employs spatial intersection logic to remove NoData and invalid pixels, ensuring high-quality input for ML.Feature Engineering: Generates temporal features like lagged variables and rolling averages to capture drought trends over time.Predictive Modeling: Utilizes Random Forest algorithms to classify drought severity classes.🏗 Pipeline ArchitectureThe project follows a modular 4-stage pipeline:Extraction: Ingests raw .tif files and organizes them by country and data type.Transformation:  Aligns disparate grids (e.g., matching NDVI pixels to Rainfall pixels) to ensure spatial consistency.Feature Engineering:  Converts raw pixel values into predictive features, including lag, rolling means, and categorical classifications.Modeling:Trains a Random Forest model to predict drought categories based on input indices.📂 Project StructurePlaintextproject/
